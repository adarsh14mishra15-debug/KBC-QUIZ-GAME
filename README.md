# KBC-QUIZ-GAME
A KBC-style quiz game in Python with random questions and a lifeline


A Python command-line quiz game inspired by "Kaun Banega Crorepati."

## Features
- Random question selection from a question bank each playthrough
- Prize ladder from 10 Lakhs up to 10 Crores
- One 50-50 lifeline per game

## How to run
python3 "KBCQUIZ GAME.py"

## What I learned
Started as a deeply nested if-else script kbc.py, then refactored it into a loop-based
design using lists, dictionaries, and randomization to make it cleaner and easier
to extend.
